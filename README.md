# Termux Toolbox

Small Termux helper collection for clean command runs, log capture, copy-window output and repeatable Android/Pixel shell workflows.

## Scope

This repository is intended to contain public, reusable helper scripts and documentation only.

It must not contain:

- SSH keys
- API tokens
- rclone configs
- real credentials
- private logs
- private network inventory
- device-specific secrets

## Install

```sh
git clone https://github.com/Lycidias93/termux-toolbox.git
cd termux-toolbox
bash ./install.sh
```

## Verify

```sh
bash ./verify/verify-termux-toolbox.sh
```
